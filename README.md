# sigtime

돌연변이 시그니처와 유전자 발현량의 시계열 연관 분석 도구

```
돌연변이 시그니처와 유전자 발현량의 시계열 연관 분석 프레임워크
유지원, 윤정환, 조겨리 ｜한국정보과학회
```

## sigtime-server

분석도구는 웹 형태로 구현되었으며, 웹(프론트) 파트 입니다.
